Opis metod:

Do kazdej metody jako argument przekazywana jest strata. Jest to wektor złozony ze zlogarytmowanego ilorazu wartosci z danego okresu i poprzedniego.

Metoda historyczna: Metoda historyczna polega na obliczeniu zmiennej VaR na podstawie danych historycznych. W tym przypadku funkcja sapply jest używana do wyliczenia kwantyla rzędu 0.99 dla 500 dniowego okna estymacji. Następnie, dla każdego okna, wyliczany jest Expected Shortfall (ES) jako średnia ważona wartości, które przekraczają VaR. Ostatecznie, zliczana jest równiez liczba przypadków przekroczenia VaR.

Metoda historyczna z wagami:

Metoda historyczna z wagami jest podobna do zwykłej metody historycznej, ale z uwzględnieniem wag dla poszczególnych obserwacji. W tym przypadku, waga dla każdej obserwacji jest obliczana na podstawie kwantyla rzędu 0.995 i jest wyliczana dla każdego okna estymacji. Następnie, dla każdego okna, obliczany jest VaR na podstawie rozkładu dyskretnego z wykorzystaniem funkcji DiscreteDistribution i kwantyla rzędu 0.99. W kolejnym kroku obliczne jest ES jako średnia ważona wartości, które przekraczają VaR. Wartości są sortowane malejąco, a następnie wyliczane są wagi dla każdej wartości. Ostatecznie, zliczana jest rowniez liczba przypadkow przekraczajacych wartość var.

Modelowanie z wykorzystaniem EWMA: Dla każdej chwili czasowej, wartość EWMA jest wyliczana jako pierwiastek kwadratowy z sumy wykładniczo ważonych kwadratów wartości w oknie estymacji. Następnie, dla każdego okna, wyliczana jest zmodyfikowana wartość strata_EWMA dla każdej obserwacji, aby uwzględnić zmienność. Ostatecznie, dla każdego okna 500 dni, VaR jest wyliczany jako kwantyl rzędu 0.99 dla zmodyfikowanych wartości strata_EWMA. ES jest obliczany jako średnia ważona wartości, które przekraczają VaR. Ostatecznie zliczana jest liczba przypadków przekroczenia VaR.

METODA HISTORYCZNA

metoda_historyczna <- function(strata){
  
  var <- sapply(1:(length(strata)-499), function(i){
    quantile(strata[i:(i+499)], 0.99)
  })
  
  es <- sapply(1:(length(strata)-499), function(i){
    w <- strata[i:(i+499)]
    mean(w[which(w>quantile(w,0.99))])
  })
  
  liczba_przekroczen <- sapply(1:(length(strata)-499), function(i){
    sum(strata[i:(i+499)]>var[i])
  })
  
  data.frame(X=ymd(dane[501:n,1]),VAR=var, ES=es, Liczba_przekroczen=liczba_przekroczen)
}

METODA HISTORYCZNA Z WAGAMI

metoda_historyczna_wagi <- function(strata){
  
  q <- 0.995
  waga <- q^(500-c(1:500))*(1-q)/(1-q^500)
  
  var <- sapply(1:(length(strata)-499), function(i){
    d <- DiscreteDistribution(supp = strata[i:(i+499)],prob = waga)
    qd <- q(d)
    qd(0.99)
  })
  
  es <- sapply(1:(length(strata)-499), function(i){
    straty <- data.frame(Strata=strata[i:(i+499)],Waga=waga)
    straty <- straty[order(-straty$Strata),]
    
    e <- straty[which(straty$Strata>=var[i]),]
    e$w <- e$Waga/0.01
    e[nrow(e),"w"] <- 1-sum(e[-nrow(e),"w"])
    sum(e$Strata*e$w) 
  })
  
  liczba_przekroczen <- sapply(1:(length(strata)-499), function(i){
    sum(strata[i:(i+499)]>var[i])
  })
  
  wyniki <- data.frame(X=ymd(dane[501:n, 1]),VAR=var, ES=es, Liczba_przekroczen=liczba_przekroczen)
  wyniki
}

METODA Z WYKORZYSTANIEM EWMA

metoda_heteroskedastycznosc_EWMA <- function(strata, lambda){
  
  EWMA <- var <- es <- liczba_przekroczen <-  0
  
  EWMA[1] <- sd(strata[1:500])
  for(i in 1:(length(strata))){
    EWMA[i+1] <- sqrt(lambda*(EWMA[i]^2)+(1-lambda)*(strata[i]^2))
  }
  
  for(i in 1:(length(strata)-499)){
    strata_EWMA <- 0
    ewma <- EWMA[i:(i+500)]
    st <- strata[i:(i+499)]
    for(j in 1:500){
      strata_EWMA[j] <- st[j]*(ewma[501]/ewma[j])
    }
    var[i] <- quantile(strata_EWMA, 0.99)
    es[i] <- mean(strata_EWMA[which(strata_EWMA>quantile(strata_EWMA,0.99))])
    liczba_przekroczen[i] <- sum(strata[i:(i+499)]>var[i])
  }
  
  wyniki <- data.frame(X=ymd(dane[501:n,1]),VAR=var, ES=es, Liczba_przekroczen=liczba_przekroczen)
  wyniki
}

Test Kupca Test Kupca to jeden z testów wstecznych sprawdzający jak często w historii strata przekroczyła wyznaczony VaR. Weryfikuje on, czy częstość występowania wyjątków (przekraczania wartości VaR) jest zgodna z odpowiednim kwantylem VaR.

H0: VaR jest dobrze wyznaczony

H1: VaR jest źle wyznaczony

Weryfikacja hipotezy: zliczenie liczby przekroczeń i sprawdzenie, czy liczba ta mieści się w odpowiednim przedziale.

Tabela z przedziałami, w których powinna znaleźć się liczba przekroczeń przy zadanej długości okna estymacyjnego i dla danego poziomu VaR, aby można było uznać hipotezę zerową w teście Kupca za prawdziwą.
